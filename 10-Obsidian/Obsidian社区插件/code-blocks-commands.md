## 概述

### 插件名称
Code Blocks commands

### 主要功能
- 提供插入带标记代码块的命令。
- 支持使用反引号触发命令，调出标记选择菜单。
- 能根据使用习惯对标记列表进行排序。
- 允许自定义语法高亮标记。

### 适用场景
适用于在Obsidian中需要频繁插入代码块并进行语法高亮标记的场景，如程序员记录代码笔记、技术人员撰写技术文档等。

### 核心特色
- 操作便捷，通过反引号即可快速触发标记选择。
- 标记列表排序功能可根据使用习惯优化，提高操作效率。
- 支持自定义标记，满足多样化的语言需求。

### 使用建议
- 初次使用时，可先熟悉插入代码块和触发标记选择菜单的操作。
- 根据自身需求，自定义常用的语法高亮标记，提升使用体验。
- 利用标记列表排序功能，优先显示常用标记。
