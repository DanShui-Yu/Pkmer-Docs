## 概述

**插件名称：Inline Code Copy**

该插件允许用户在阅读视图中通过简单点击快速复制行内代码块的内容，极大地提升了便利性。尤其适合存储凭证和账户信息的用户，避免了传统复制方式中需要手动选择文本和按Ctrl + C的繁琐过程。使用该插件后，只需点击代码块，即可轻松复制内容，提升工作效率。安装过程简单，只需将相关文件复制到Obsidian插件目录中即可。
